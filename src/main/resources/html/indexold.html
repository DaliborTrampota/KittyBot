Cute commands

You can use .cuddle<@user> .hug<@user> .pat<@user> .kiss<@user> .slap<@user> .feed<@user> .tickle<@user> .poke<@user> to send cute GIFs to them!

Image Search

You can search for images by using .search <text>

Self-assignable roles

You can use .roles add <@role> to add and .roles remove <@role> to remove roles. You are able to set this in the webinterface too.
To assign the roles use .roles and click the reaction emotes!
